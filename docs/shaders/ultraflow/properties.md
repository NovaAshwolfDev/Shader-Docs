# Ultra Flow Shader Parameters

| Property | Type | Description | Effects |
|-----------|------|-------------|--------|
| Sticker | Texture | The Icon that is used for the Wave Effect. | None |
| Shape Type | Drop Down | The Type of Shape that is used for the Wave Effect | None, Circle, Hexagon, Triangle, Alpha Mask |
| Color | Color | The Solid Color of the Wave Effect | None |
| Use Gradient | Checkbox | Uses the Gradient Colors for the Wave Effect | None |
| Use Rainbow | Checkbox | Uses a Diagonal Rainbow for the Wave Effect | None |
| Parallax Depth | Checkbox | Enables Parallax Depth for the Wave Effect | None |
| Soft Shape Strength | Slider | The Strength of the Softness Effect | Applys to all Shape Effects |
| Soft Shape Softness | Slider | The Softness of the Softness Effect | Applys to all Shape Effects |
| Shape Mask | Texture | The Alpha Mask for the Shape Effect | Applys to the Alpha Mask Type in the Shape Effect Dropdown |

# Ultra Flow Shader

The Ultra Flow Shader is a Shader that works by using a Directional Wave, a Texture and a Solid Color, Gradient or Rainbow to create a flowing color effect.
<img src="https://cdn.miloashwolf.gay/file/storage-miloashwolf-gay/UltraFlow.png" alt="Ultra Flow Shader" width="50%"/>
## How to use

1. To use this shader, you'll need to input your license key in Window that pops up when you first import the shader package.
2. Put some textures in the material slot OR use the template built in!
3. Enjoy!
